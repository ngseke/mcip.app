## Recommended Environment

- Node.js `>=18`
- pnpm `>=8` (do not use **npm** or **yarn**)

## Development Commands
### Setup

```bash
pnpm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm run dev
```

### Production

Build the application for production:

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm run preview
```
